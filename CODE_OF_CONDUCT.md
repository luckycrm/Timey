# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume good intent
- Welcome newcomers and different viewpoints
- Give actionable technical feedback

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, insults, or derogatory comments
- Publishing private information without permission
- Any conduct that is inappropriate in a professional setting

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may take corrective action for unacceptable behavior.

## Reporting

If you experience or witness unacceptable behavior, open a private security report or contact maintainers directly via GitHub.
