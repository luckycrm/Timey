# Support

## Getting Help

- Use GitHub Issues for bugs and feature requests
- Use GitHub Discussions for Q&A and implementation ideas

## Before Opening an Issue

- Confirm you are on the latest `main`
- Include clear repro steps
- Include logs/screenshots where possible
- Share environment details (OS, Bun version, browser)

## Project Scope

Timey is an open-source employee, project, and time management platform under active development. Suggestions are welcome.
